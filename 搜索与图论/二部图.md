## 染色法判定二部图
### 原理
* 一个图是二部图的**充要条件**是：不存在奇数环、可以被2染色
* 通过用**两种颜色**对每个联通图染色，可以判定其是否为二部图
* 使用DFS进行染色和判断
* 时间复杂度为 $O(m+n)$ 

### 核心代码
* 使用DFS染色，并判断是否有矛盾
```
bool dfs(int u, int c){
    color[u] = c;
    for(int i=h[u];i!=-1;i=ne[i]){
        int t = e[i];
        if(!color[t]) {
            if(!dfs(t, 3-c)) return false;
        }
        else if(color[t]==c) return false; //矛盾
    }
    return true;
}
```
